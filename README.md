# Dat Examples

You can build awesome applications with Dat too! These examples show how the core modules work and various ways to use Dat. 

If you want to customize how Dat works in your application, these examples will get you started. If you want to plug a Dat module into your app, the [dat-js](https://github.com/joehand/dat-js) module exposes the basic functionality of Dat, sharing and downloading files.
